Topic: Heart Disease Prediction Using Logistic Regression and Support Vector Machines
Abstract: Heart disease is one of the most common and serious health conditions worldwide, and early prediction can help patients receive medical attention sooner. In practice, determining whether a patient is at risk is not always simple because there are many affecting factors like age, blood pressure, cholesterol, and etc. The goal of this project is to develop a machine learning model that predicts whether a patient has heart disease based on clinical data.
The motivation for this project is that heart disease prediction is well suited for machine learning. A simple rule based approach may focus too much on one feature and fail to capture the combined effect of multiple variables. Machine learning offers a more systematic way to learn patterns from data and make predictions using several patient attributes together. It also allows different models to be compared in a clear and measurable way.
This project will use a publicly available heart disease dataset and treat the task as a binary classification problem. A basic baseline, such as predicting the majority class, will first be established. Then, logistic regression and support vector machines will be trained and compared. Logistic regression offers a simple and interpretable approach, while SVM may better capture more complex patterns in the data. The project will include data preprocessing, feature analysis, model training, and evaluation using metrics such as accuracy, precision, recall, and F1 score. The aim is to examine how machine learning can improve prediction beyond a naive baseline.
